How this reads

Site details and every yellow-cell input from the original workbook live on the Inputs sheet. From there, Setback Calc derives your buildable footprint from the NBC baseline setback table plus the height-linked increment rule. Height & FAR checks your proposed massing against the road-width height rule and your FAR envelope. Hotel Program plans keys per floor from the room module and core areas, and builds up guestroom, FOH and BOH area. Summary rolls it all into a single verdict.

Important caveat. NBC 2016 is a model / advisory code (Part 3 = Development Control Rules). It only becomes legally binding once adopted into your State or local Building Bye-Laws / DCR (UDCPR–Maharashtra, TNCDBR–Tamil Nadu, Karnataka BBMP/KMC, Delhi UBBL, and so on), which frequently set stricter or different setback, FAR, parking and height-vs-road-width values than this baseline. Every figure here is an indicative, editable baseline — replace it with your local sanctioning authority's values before using this for submission or investment decisions. Fire-access, structural, MEP and lift-code requirements are simplified as area allowances and must be verified against NBC Part 4 / Part 8, relevant IS codes, and a licensed architect's design.
